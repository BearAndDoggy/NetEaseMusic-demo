## 网易云音乐

[点击使用](https://bearanddoggy.github.io/NetEaseMusic-demo/src/index.html)

###前端页面：

* 1: 推荐音乐、热歌榜、搜索（只有界面）
* 2: 首页歌单、最新音乐
* 3: 播放页面：黑金胶带滚动、歌词滚动高亮功能
* 4: 歌单 -> 歌曲列表
* 5: Flex与Vw实现，移动端使用更佳

###管理页面
* 1: 歌曲上传、新建歌曲、歌曲编辑
* 2: 歌单创建、歌单编辑

###后台
* 1: 基于leanCloud与七牛网

###语言
* 1: jQuery、原生JavaScript、HTML5、CSS3
